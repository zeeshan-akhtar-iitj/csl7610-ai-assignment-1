# CSL7610 Artificial Intelligence - Assignment 1

This repository contains the Java codebase for Assignment 1, demonstrating the implementation of State Space Search algorithms and Constraint Satisfaction Problem (CSP) solvers.

## Project Structure
~~~text

├── src/
│   ├── q1_search/           
│   │   ├── State.java             # 8-Puzzle state representation
│   │   ├── Node.java              # Search tree node structure
│   │   ├── SearchAlgorithms.java  # BFS, DFS, A*, IDA*, Simulated Annealing
│   │   └── MainQ1.java            # Controller & Successor Function
│   ├── q2_csp/

├── input.txt                      # Centralized input file for Q1 and Q2
~~~

## Part 1: State Space Search (8-Puzzle Manuscript Sorter)
The first part of the assignment solves an 8-puzzle variant using various search algorithms.
* **Uninformed Search:** Breadth-First Search (BFS), Depth-First Search (DFS)
* **Informed Search:** A* (with Misplaced Tiles and Manhattan Distance heuristics)
* **Memory-Bounded & Local Search:** Iterative Deepening A* (IDA*), Simulated Annealing

## Part 2: Constraint Satisfaction Problem (Security Bot Scheduler)
The second part schedules 3 security bots (A, B, C) across 4 time slots (Slot1 to Slot4) subject to:
* **Unary Constraints:** Maintenance downtime (e.g., Slot4 != C)
* **Binary Constraints:** No back-to-back shifts for the same bot.
* **Global Constraints:** Every bot must be used at least once.
* **Techniques Used:** Backtracking Search, Minimum Remaining Values (MRV) heuristic, and Forward Checking inference.

## How to Execute the Code

### Prerequisites
* Java Development Kit (JDK) 11 or higher.
* An IDE like IntelliJ IDEA or Eclipse (Recommended).

### Execution Steps
1. Clone this repository to your local machine.
2. Ensure `input.txt` is located in the root directory of the project.
3. **To run Question 1 (Search):** Execute the `main` method in `src/q1_search/MainQ1.java`.
4. **To run Question 2 (CSP):** Execute the `main` method in `src/q2_csp/CSPMain.java`.
